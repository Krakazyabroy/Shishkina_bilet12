# Шишкина Юлия ИС-22-1
## Калькулятор стоимости жалюзи
WPF-приложение для расчёта стоимости жалюзи и генерации PDF-чека.

# Возможности
Ввод ширины и высоты (в метрах)
Выбор материала: Алюминий (15,50 руб./м²) или Пластик (9,90 руб./м²)
Расчёт итоговой стоимости
Ввод суммы оплаты и расчёт сдачи
Формирование PDF-квитанции
